To get your website live on a custom URL, follow these steps:

### 1. Push to GitHub
1. Create a new repository on [GitHub](https://github.com) named `good-vibes-barber`.
2. Upload all the files in this directory to the `main` branch.

### 2. Connect to Vercel (Recommended)
Because this app uses React and ESM modules, **Vercel** is the easiest way to host it for free:
1. Go to [Vercel](https://vercel.com) and sign up with your GitHub account.
2. Click **"New Project"**.
3. Select your `good-vibes-barber` repository.
4. If asked for a "Framework Preset," select **Vite** or **Other**.
5. Click **"Deploy"**.

### 3. Add your Gemini API Key
To make the AI Assistant work on the live site:
1. In your Vercel Dashboard, go to **Settings** > **Environment Variables**.
2. Add a new variable:
   - **Key**: `API_KEY`
   - **Value**: `[Your-Google-Gemini-API-Key]`
3. Redeploy the site.
